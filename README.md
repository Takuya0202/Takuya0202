## main product 😎😎😎
### 飲食店検索アプリ 「foodsnap」(現在も開発中)
メニューを中心にアプローチし、直感的に料理の魅力を伝える飲食店検索アプリ <br>
従来の飲食店検索アプリにあるカード形式の一覧表示や複雑な店舗情報を廃止し、<br>
メニュー中心にアプローチし、直感的に店舗の魅力を伝え、フリック操作で飲食店を発見できるwebアプリケーションです<br>
